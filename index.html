Measuring Urban Nature Accessibility

Visualising the 3-30-300 Guideline for Amsterdam West

An interactive geospatial analysis of Amsterdam West's compliance with the 3-30-300 urban greening guideline. The project combines visibility analysis, satellite-derived vegetation metrics and network accessibility modelling to evaluate how effectively neighbourhoods provide access to urban nature.

Tools & Data

Leaflet VS Code Python Sentinel-2 OpenStreetMap Tree Dataset

Objective

Identify buildings that meet one or more of the 3-30-300 criteria and highlight areas that may benefit from additional greening.

Method

Combined tree visibility, canopy cover and walking accessibility analyses using Python and geospatial datasets.

Outcome

Produced an interactive map showing compliance with the 3-30-300 guideline across Amsterdam West.

Mapping Environmental Inequality

Houston's Heat Divide

This project explores how urban heat exposure varies across Harris County, Texas. By combining satellite imagery with socio-economic data, the study examines the links between vegetation cover, surface temperature and income levels across different neighbourhoods.

Tools & Data

ArcGIS StoryMaps ArcGIS Pro Landsat Census Data NDVI LST

Objective

Examine how heat exposure varies across neighbourhoods and identify links between temperature, vegetation and income.

Method

Combined Landsat-derived LST and NDVI data with census data to analyse spatial patterns across Harris County.

Outcome

Revealed that wealthier neighbourhoods generally had more vegetation and lower temperatures than less affluent areas.

Finding the Optimal Route

Proposed Industrial Wastewater Pipeline Route in the Maas-Waal Region

A spatial suitability and routing analysis designed to identify the most suitable path for a new industrial wastewater pipeline while minimising environmental, topographic and land-use constraints.

Tools & Data

ArcGIS Pro ModelBuilder AHN Elevation Land Use Cost Distance

Objective

Identify the most suitable route for a new industrial wastewater pipeline while meeting environmental and planning constraints.

Method

Developed a routing workflow in ArcGIS ModelBuilder using suitability analysis, cost-distance modelling and least-cost paths.

Outcome

Produced a repeatable model capable of identifying efficient and compliant pipeline routes.

Machine Learning for Rapid Flood Detection

Comparing Global Thresholding and Random Forest Classification for Flood Mapping

This project compares a traditional threshold-based approach with a Random Forest model for flood mapping using Sentinel-1 radar imagery. The 2023 Kakhovka Dam breach was used as a real-world case study to evaluate which approach produced more reliable flood extent maps.

Tools & Data

Google Earth Engine ArcGIS Pro Sentinel-1 Random Forest

Objective

Evaluate whether machine learning can improve flood mapping accuracy compared with a traditional threshold-based approach.

Method

Classified Sentinel-1 imagery using Random Forest and Global Thresholding, then compared both outputs against a reference dataset.

Outcome

Random Forest produced more accurate flood extent maps and reduced classification errors across complex landscapes.

Designing Climate-Resilient Neighbourhoods

From Grey to Green: A Climate-Adaptive Redesign of De Buurt, Wageningen

A redesign proposal for a neighbourhood in Wageningen aimed at reducing heat stress, improving biodiversity and increasing access to green space through nature-based solutions and sustainable urban design.

Tools & Data

ArcGIS Pro QGIS Canva Sentinel-2 Multi-Criteria

Objective

Improve climate resilience, biodiversity and access to green space while maintaining a liveable neighbourhood environment.

Method

Applied GIS-based Multi-Criteria Analysis to identify priority intervention areas and guide the redesign process.

Outcome

Produced a practical redesign strategy that enhances cooling, biodiversity and green infrastructure throughout the neighbourhood.
