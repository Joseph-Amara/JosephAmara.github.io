🌳 Urban Forestry Analysis

Visualizing the 3-30-300 guideline for Amsterdam West

This project maps the 3-30-300 green space guideline for Amsterdam West, highlighting buildings that meet criteria for visible trees, canopy cover, and proximity to parks.

Python VS Code Leaflet Sentinel-2 OSM
Objective
To create an interactive visualisation assessing building-level compliance with the 3-30-300 urban forestry guideline.
Method
A spatial model was built using building footprints, tree locations, and Sentinel-2 imagery. Sightline simulations, NDVI calculations, and network analyses were combined to classify each building, mapped interactively via Leaflet.

🌡️ Urban Heat Island Study

Houston's Heat Divide

This project investigates the environmental inequality of the urban heat island effect across Harris County, Texas, linking vegetation density, surface temperatures, and socioeconomic status.

ArcGIS StoryMaps Landsat 7/8 Socio-demographic Data
Objective
To highlight disparities in urban heat distribution and provide local government with insights for fair, climate-resilient policies.
Method
Land Surface Temperatures (LST) and vegetation density (NDVI) derived from Landsat data were overlaid with income demographics. The results, demonstrating that wealthier neighbourhoods are greener and cooler, were visualised in a StoryMap.

🚰 Spatial Routing Model

Proposed New Pipeline Route for Industrial Wastewater in the Maas Waal Area

This spatial decision-making project identifies the most suitable path for a new industrial wastewater pipeline based on environmental and infrastructural constraints.

ArcGIS Pro Land Use Data AHN Topography
Objective
To determine the optimal, compliant route for a wastewater pipeline connecting a source to its destination across a complex landscape.
Method
The routing analysis was fully automated using ArcGIS Pro ModelBuilder. Geoprocessing features like spatial overlays and cost-distance calculations were chained to evaluate terrain and land use, resulting in a reproducible routing model.
🌊 Flood Extent Mapping

Comparing Global Thresholding and Random Forest Classification for Improved Flood Mapping Using Sentinel-1 Data

Using the 2023 Kakhovka Dam breach as a case study, this project evaluates machine learning versus traditional methods in identifying flooded areas across complex landscapes.

Google Earth Engine ArcGIS Pro Sentinel-1 (SAR)
Objective
To compare the accuracy of a traditional Global Thresholding approach against a Random Forest classification model for mapping flood extents.
Method
Pre-flood and during-flood radar images were processed. A Random Forest classifier and Global Threshold method were applied to distinguish water from land, and error rates were calculated against a high-resolution reference map.

🏙️ Climate-Adaptive Urban Design

From Grey to Green: A Climate-Adaptive Redesign of De Buurt, Wageningen

This project proposes a climate-adaptive redesign of a neighbourhood vulnerable to heat stress, integrating nature-based solutions to create a biodiverse community space.

ArcGIS Pro QGIS Canva Sentinel-2
Objective
To improve the neighbourhood's climate resilience, biodiversity, and access to public green space through a strategic redesign balancing environmental goals with resident needs.
Method
A Multi-Criteria Analysis pinpointed areas most in need of green interventions. A spatial design was then developed featuring new green corridors, bioswales, and wildflower meadows, visualised via a detailed poster.
